# 📘 SSC 스파르타 학습관리 시스템 — 직원 사용 안내

> 이 문서는 **신규 입사 직원 및 현장 운영팀**을 위한 통합 매뉴얼입니다.
> 학생 관리, 출결, 진도, 소통, 정산까지 한 시스템 안에서 처리합니다.
> 처음이라면 **① 개요 → ② 로그인 → ③ 매일 하는 업무** 순서로 읽어보세요.

---

## 1. 한눈에 보기

SSC 관리 시스템은 **공무원 직렬(경찰·소방 등) 수험생을 위한 스파르타 학습관실 운영 도구**입니다.
크게 세 개의 화면으로 나뉩니다.

| 화면 | 누가 쓰나 | 주소 |
| --- | --- | --- |
| **관리자 대시보드** | 직원·운영팀 | `/admin` |
| **등하원 키오스크** | 학생(현장 태블릿) | `/attend/kiosk` |
| **학생 리포트(마이페이지)** | 학생 본인·학부모 | `/student` |

> 💡 직원이 주로 쓰는 곳은 **관리자 대시보드** 한 곳입니다. 키오스크와 학생 리포트는 학생이 직접 사용합니다.

---

## 2. 로그인 & 기본 개념

### 로그인
1. 브라우저에서 **`/admin`** 접속
2. 발급받은 **아이디 / 비밀번호** 입력 → 자동으로 홈 대시보드로 이동
3. 계정은 **관리자 계정 관리** 메뉴에서 슈퍼 관리자가 발급합니다.

### 꼭 알아야 할 개념

- **센터(캠퍼스)**: 원주 / 춘천 / 충주 등 지점 단위. 상단 바에서 센터를 전환해 해당 지점 학생만 봅니다.
  - 일반 직원 계정은 **본인 소속 센터만** 보이고, 슈퍼 관리자는 전 센터를 봅니다.
- **권한(role)**: 슈퍼 관리자에게만 **'관리자 계정 관리'** 메뉴가 보입니다.
- **진도(순공)**: 학생의 학습 데이터는 한 곳(단일 소스)에서 관리됩니다. 여러 화면에 같은 값이 연동되니 **한 곳에서만 수정**하세요.

### 메뉴 여는 법
- 좌측 상단 **☰ (햄버거)** 버튼 → 전체 메뉴
- PC에서는 메뉴 상단의 **📌 고정** 버튼으로 좌측 사이드바를 항상 켜둘 수 있습니다.

## 4. 학생은 무엇을 보나요?

학생은 **`/student`** 로 로그인하면 본인 리포트(마이페이지)가 열립니다. 직원이 입력한 데이터가 학생 화면에 그대로 반영되니 흐름을 알아두면 좋습니다.

학생 리포트 탭 구성:
- **홈 개요** — 오늘 할 일·출결·공부시간 요약
- **과목별 진도** — 직원이 관리하는 진도가 표시
- **성적 분석** — 모의고사 성적 추이
- **실행 계획 / 시간표** — 학습 플랜
- **알림** — 직원이 보낸 메시지에 학생이 답장(양방향)
- **상담 / 벌점** — 상담 기록과 벌점 내역 열람
- **부가 기능** — 뽀모도로 타이머, 쿠폰 교환, 미션 진행도, 도시락·OT·모의고사·행사 안내 알림

> 💡 학생이 **휴가·도시락·OT·모의고사 불참** 등을 신청하면 → 직원의 **통합 인박스** 또는 해당 메뉴에 승인 대기로 들어옵니다.

## 5. 자주 하는 업무 시나리오

### ✅ 매일 아침 루틴
1. **홈 대시보드** 확인 — 오늘 출결·신청·알림 현황
2. **통합 인박스** 확인 — 밤사이 들어온 학생 문의·신청 처리
3. **출결 상세 / 좌석 현황판** — 등원 현황 점검

### ✅ 신규 학생이 등록할 때
1. **학생 추가** 로 기본 정보 등록
2. **원생 종합 관리** 에서 진도·결제 등 상세 입력
3. (필요 시) **OT 참여 관리** 에서 오리엔테이션 배정

### ✅ 학생이 휴가를 신청했을 때
1. **통합 인박스** 또는 **휴가 쿠폰 관리** 에 승인 대기 표시
2. 월 한도·쿠폰 잔여 확인 후 승인/반려
3. 결과는 학생 알림으로 자동 전달

### ✅ 도시락 운영
1. **도시락 신청** 에서 주차별·끼니별 집계 확인
2. **A4 인쇄표** 출력해 주방 전달
3. 마감/정산 처리, 추가 신청은 승인제

### ✅ 학생을 칭찬/제재할 때
- **벌점 · 상점 관리** 에서 부여 → 학생 리포트에 자동 반영

### ✅ 동기부여(미션/랭킹)
- **쿠폰 미션 설정** 으로 목표 달성 시 쿠폰 자동 지급
- **순공 랭킹** 으로 학습 분위기 독려

## 6. 주의사항 & 팁

- ⚠️ **센터 확인 먼저**: 작업 전 상단 바의 센터가 맞는지 항상 확인하세요. 다른 센터 학생을 잘못 수정하지 않도록.
- ⚠️ **진도/순공은 한 곳에서만 수정**: 여러 화면에 연동되므로 중복 입력 금지.
- ⚠️ **승인 대기 방치 금지**: 휴가·도시락·OT·모의고사 신청은 학생이 결과를 기다립니다. 인박스를 매일 비우세요.
- 💡 **메시지 템플릿** 활용: 반복 안내는 메시지 발송의 공유 템플릿으로 빠르게.
- 💡 **키오스크**는 현장 태블릿 전용 — 직원 PC에서 열어두지 마세요.
- 🔐 **로그아웃**: 공용 PC에서는 좌측 메뉴 하단 **로그아웃**으로 반드시 종료.

## 7. 문제가 생기면

| 상황 | 조치 |
| --- | --- |
| 로그인이 안 됨 | 슈퍼 관리자에게 계정/비밀번호 재발급 요청 |
| 메뉴가 안 보임 | 권한(센터/슈퍼) 문제일 수 있음 — 관리자에게 문의 |
| 데이터가 안 맞음 | 센터 전환 상태 먼저 확인 후 운영팀에 보고 |
| 키오스크 오류 | 태블릿 새로고침 후 재시도, 지속되면 운영팀 보고 |
